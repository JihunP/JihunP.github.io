Project FFPC

Full-Stack Dev

A need to have better designed projects

In this project I wanted to experiment with 3D models and with technologies such as Three.js which allow developers to create 3D scenes on websites.

Three.js is a powerful JavaScript library used for creating and displaying stunning 3D graphics in web applications. Leveraging the capabilities of WebGL, Three.js simplifies the process of developing immersive 3D experiences by providing a comprehensive set of tools and utilities.

There is more than one model being used. In this scenario, the model was imported from a pre-made blender file.

Another part of the project I wanted to highlight was the Google Maps API.

Utilizing Google Maps and their comprehensive documentation, I was able to create a custom marker for the location of the church.

I can embed dynamic maps into my web applications, enabling users to interact with maps, pan, zoom, and switch between different map views. This interactivity enhances the user experience and provides users with flexible ways to explore geographic information.
